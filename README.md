
# S.T.A.R.L.I.G.H.T. — Steele Tech
A Life‑Integrated AI Timer for Growth & Harmony

## Features
- Boot animation + soft chime + voice intro
- MIA (calm female) & BRIAN (analytic) voices
- Animated orb + speech‑synced rings
- Presets (2m/5m/10m/25m/45m) and custom timers
- PWA (installable) + GitHub Pages workflow

## Deploy (GitHub Pages)
1. Create public repo (e.g., `starlight-web`).
2. Upload all files at repo root (keep `.github/workflows/deploy.yml`).
3. Settings → Actions → Allow all actions.
4. Settings → Pages → Source = GitHub Actions.
5. Commit to `main` → Actions deploys → open `https://USERNAME.github.io/REPO/`.

## Brand
- Replace `assets/logo.svg` with your logo (same filename).
- Replace `icons/icon-192.png` and `icons/icon-512.png` with app icons.
